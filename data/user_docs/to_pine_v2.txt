User Manual / Migration guides / To Pine Script® version 2 To Pine Script ® version 2 Pine Script version 2 is fully backwards compatible with version 1. As a result, all v1 scripts can be converted to v2 by adding the //@version=2 annotation to them. An example v1 script: study(&quot;Simple Moving Average&quot;, shorttitle=&quot;SMA&quot;) src = close length = input(10) plot(sma(src, length)) The converted v2 script: //@version=2 study(&quot;Simple Moving Average&quot;, shorttitle=&quot;SMA&quot;) src = close length = input(10) plot(sma(src, length)) Previous To Pine Script® version 3 Pine Q&A chat Stack Overflow ↗ Telegram ↗ Reddit ↗ Copyright © 2026 TradingView, Inc.
